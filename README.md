🚀 Promotion & Discount Management Module
Athukorala Traders - Industrial Inventory System
This module is a core part of the Athukorala Traders ecosystem. It allows the manager to create, update, and monitor all price discounts for industrial tools in the Kaduwela sector.

🛠️ Tech Stack Used
Frontend: React.js with Tailwind CSS (Glassmorphism UI).

Backend: Java Spring Boot (REST API).

Database: MySQL (Relational Data).

Icons: Lucide-React.

✨ Main Features (CRUD)
Create: Add new discount campaigns with a name, percentage, and dates.

Read: View all active and past promotions in a cinematic dashboard.

Update: Change the discount amount or extend the expiry date easily.

Delete: Remove a promotion if the sale is over.

🏗️ How it Works
Discount Engine: The Java backend calculates the new price and sends it to the React frontend.

Live Sync: When a discount is added, a "Discount🔥" tag appears on the product card automatically.

Crash-Proof Loading: The system uses a special "Try-Catch" logic to ensure the app never crashes if an image is missing.
